# Employee Security Portal

Employee Security Portal is a lightweight web-based application that simulates an internal enterprise portal for managing employee information, system status, and patch management activities.

The project is built using HTML, CSS, and JavaScript and demonstrates a modular front-end architecture suitable for internal administrative dashboards.

## Features

- Employee Dashboard
- Employee Directory
- Patch Management
- System Status
- Reports
- Settings
- Responsive User Interface

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)

```
employee-security-portal-sast-lab/

```

## Getting Started

Clone the repository and open `index.html` in your browser.

```bash
git clone https://github.com/<your-username>/employee-security-portal-sast-lab.git
cd employee-security-portal-sast-lab
```

No installation or build process is required.
